WordPress vs Dashboard price audit

Tool

Upload the WordPress trips export. This fetches the live dashboard feed, matches trips by ship + departure date, and compares each cabin by name, flagging price differences, discounts shown on one side but not the other, and sold-out disagreements. Cabins whose names don’t align between the two systems are listed separately under “naming needs mapping” rather than reported as errors. Download the full report as CSV or Markdown.
